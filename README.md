# Overview

My son's Kindergarden teacher asked him to make up words from the
letters in "elephant and pigge".  He had a couple of good ones and I
couldn't think of any.

So I made this.

# Usage

    $ cabal install
    $ elephant-pig ElephantAndPigge </path/to/dictionary
    [...]
    dentine
    denting
    deplane
    deplaning
    deplete
    [...]
